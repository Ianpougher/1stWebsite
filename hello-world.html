CS2610 Basic Linux Shell commands

file meta data commands

ls: directory display

the "ls" command allows you to display the current directory's contents in the terminal

example:

username:directorylocation $ ls -l

total 8

-rw-rw-r-- 1 ubuntu ubuntu 907 Nov 1 05:45 README.md

-rw-rw-r-- 1 ubuntu ubuntu 1354 Jan 24 16:09 hello-world.html


tree: display of files in a visual format

the "tree" command dispays the active directory in

example:

Username:~/directory $ tree

├── README.md

├── examplefile.txt

├── folder1

├── hello-world.html

└── newfile.txt

1 directory, 4 files


File: denotion of file information

the "file" command denotes any information about the specific file

example:

username:~/directorylocation $ file newfile.txt

newfile.txt: empty


Basic Help Commands

Type: notation of file type

the "Type" command denotes the specific files type

example:

username:~/directorylocation $ type ls

ls is aliased to `ls --color=auto -F'


cd: basic directory accessing and movement through the file system

the "cd" command allows you move throughout the file system fo your computer

username:directorylocation $ ls -l

-rw-rw-r-- 1 ubuntu ubuntu 907 Nov 1 05:45 README.md

drwxr-xr-x 2 ubuntu ubuntu 4096 Jan 24 16:19 folder1/

-rw-rw-r-- 1 ubuntu ubuntu 1354 Jan 24 16:09 hello-world.html

username:directorylocation $ cd folder1

username:directorylocation/folder1 $

Creation, Copying, and Deletion

mkdir: creation of a new directory

the mkdir command creates a new directory or folder in the active directory location

username:directorylocation $ ls -l

total 8

-rw-rw-r-- 1 ubuntu ubuntu 907 Nov 1 05:45 README.md

-rw-rw-r-- 1 ubuntu ubuntu 1354 Jan 24 16:09 hello-world.html

username:directorylocation $ mkdir folder1

username:directorylocation $ ls -l

-rw-rw-r-- 1 ubuntu ubuntu 907 Nov 1 05:45 README.md

drwxr-xr-x 2 ubuntu ubuntu 4096 Jan 24 16:19 folder1/

-rw-rw-r-- 1 ubuntu ubuntu 1354 Jan 24 16:09 hello-world.html


cp: copying a file

the cp command copys the denoted file or directory

Username:~/directorylocation $ ls -l

total 12

-rw-rw-r-- 1 ubuntu ubuntu 907 Nov 1 05:45 README.md

-rw-r--r-- 1 ubuntu ubuntu 0 Jan 25 21:56 examplefile.txt

drwxr-xr-x 2 ubuntu ubuntu 4096 Jan 24 16:48 folder1/

-rw-rw-r-- 1 ubuntu ubuntu 3564 Jan 25 19:18 hello-world.html

Username:~/directorylocation $ cp examplefile.txt newfile.txt

rm: removal of files

the rm command deletes the denoted file

example:

username:~/workspace $ ls -l

total 16

-rw-rw-r-- 1 ubuntu ubuntu 907 Nov 1 05:45 README.md

-rw-r--r-- 1 ubuntu ubuntu 0 Jan 25 21:56 examplefile.txt

-rw-rw-r-- 1 ubuntu ubuntu 5755 Jan 27 18:35 hello-world.html
